# Visual provenance

## Quantitative figures

`model_performance_context.png` and `final_tcell_confusion_matrix.png` are
generated by `make_overview_figures.py` from committed CSV/JSON evidence. The
performance chart compares different cohorts and tasks for context only.

## UMAP panels

The main repository README embeds two archived outputs:

- `archive/prior_nsclc_workflow/figures/embeddings/stage1_umap_celltype_disease.png`
- `archive/prior_nsclc_workflow/figures/embeddings/stage2_umap_celltype_disease.png`

They were generated from the earlier Stage 1 cell-type and Stage 2 disease
fine-tuned models. They are not embeddings from the currently running
three-state T-cell perturbation screen.

## Perturbation illustration

`tcell_in_silico_perturbation.png` is an artistic scientific concept image. It
depicts a T cell, ranked gene-token deletion, model embedding recomputation, and
movement toward three possible disease-state references. It contains no
quantitative result and must not be interpreted as a measured embedding map.
